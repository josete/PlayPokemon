Installation

Requirements

Node js (v. 6.7+) download Node js from here.
Java (v. 1.8.0+) download java from here.

First step

Download PlayPokemon from the link above or here.
Uncompress the zip file, go to the destination folder and open a command line and write "npm install", this will install all the packets required for the aplication to work.

Running the aplication

In order to run the aplication open a command line and write "node app.js", this will open the configuration page in your default browser.
