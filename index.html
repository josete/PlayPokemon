Installation

Requirements

Node js (v. 6.7+) download Node js from here.
Java (v. 1.8.0+) download java from here.

First step

Download PlayPokemon from the link above or here.
Uncompress the zip file, go to the destination folder and open a command line and write "npm install", this will install all the packets required for the aplication to work.

Running the aplication

In order to run the aplication open a command line and write "node app.js", this will open the configuration page in your default browser.

Configuration

Change button names and Spark cost

First you have to open the section related to the button you want to change. Then you can write the new name in the box named Name, and the cost in the box named Cost. This changes will appear when you click out of the box. To save the changes click in the "Save" button.

Log in Mixer

To launch the aplication click in the button "Launch", this will ask you for permissions in your Mixer account, accept, you will be redirected to the main page. Congratulations the aplication is running in your channel!!.

Automatic Control

This option allows the aplication to control your emulator, if you not play on emulator or you are using a GBA emulator, please disable it.

How to use

Now you have your aplication running on your channel but yuor viewers cant do anything yet (the buttons are disabled),
to enable the buttons and start the "Poll", once you enabled the buttons your viewers have 10 seconds to vote, then you will show a message in the page with the result.

Notes
